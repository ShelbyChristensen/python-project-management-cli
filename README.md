# python-project-management-cli

## Overview
A Python CLI application to:
- Create and manage users, projects, and tasks  
- Mark tasks as complete  
- Persist data using JSON  
- Use clear CLI commands with argparse

## Features
- Add users
- Add projects
- Add tasks
- Complet tasks
